Jenkins: Continuous Integration and Automation

Jenkins is the automation server that orchestrates the entire CI/CD pipeline for building, testing, and deploying the application. Jenkins is set up to trigger automated test runs, integrate with Selenium tests, and package the application into Docker containers.

Docker: Containerization

Docker is used to package the application and its dependencies into containers, making it portable and easy to run across any environment.

Kubernetes: Orchestration and Scaling

Kubernetes handles the orchestration and management of Docker containers, ensuring that they are deployed correctly, scaled based on demand, and maintained for high availability.

Jira: Project Management and Test Planning

Jira is used as the central platform for managing the project, where all the test plans, user stories, and test cases are documented.

Selenium IDE: Test Execution

Selenium IDE is a browser-based tool used to automate the execution of test cases. We use Selenium IDE to load and execute scripts directly in the browser.

AWS: Cloud Infrastructure

The entire application is hosted on AWS, providing the scalability, performance, and reliability needed to run the application. Kubernetes runs on AWS, leveraging its infrastructure to scale and manage the application efficiently.
